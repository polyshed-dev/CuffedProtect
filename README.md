# CuffedProtect
Simple [EXILED](https://github.com/ExMod-Team/EXILED) plugin that protects players that you cuff.
### Features
- Only SCPs and the cuffer can kill a cuffed player.
- People in the cuffer's team except the cuffer can not uncuff a cuffed player, this prevents people uncuffing and shooting a player that someone else cuffed.
### Example
You are an NTF Sergeant and you cuffed Player A, who is a Class D. Only you and the SCPs can kill Player A unless you yourself uncuff him or his team (Chaos Insurgency and other Class D) releases him from the cuffs.
# Depencencies
- EXILED `9.14.2`: https://github.com/ExMod-Team/EXILED
# How to install
1. Download the latest release .dll file.
2. Put the file in your EXILED plugins folder.
3. Restart your server to activate the plugin.
